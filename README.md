# Keylogger_Python_DIO
Criação de um keylogger escrito em python para fins educacionais.
Arquivo keylogger.pyw é um keylogger simples que captura as teclas e cria um log.txt com as teclas capturadas.
Já o keylogger_email.py é um mais "perigoso" ele captura as teclas e envia automaticamente para o email indicado.
